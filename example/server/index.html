逆ハッシュ関数ツール

「ハッシュ化したい平文」にテキストを入力するとsha256でハッシュ化されます

「平文化したいハッシュ値」に16進数表記でハッシュ値を入力すると元の平文を複合することができます

ただし、平文化できるのは以下の条件に当てはまるハッシュ値のみです。

  • 平文が半角英数字に-と_を加えた64種類の文字のみからなる
  • 平文が4文字である
  • sha256でハッシュ化されたものである

平文化に失敗したら、他のハッシュ値で試してみてください。

大体81%ぐらいのハッシュ値は平文化できます。
